The Problem

Demo agents are easy.
Production agents are a different story.

🚫

No Quality Gates

Most tools let you deploy without measuring. An agent inventing policies, deleting databases, hallucinating answers — these aren't edge cases.

🔒

Vendor Lock-in

Proprietary platforms tie you to one model provider and one deployment model. Your agents become hostages to someone else's roadmap.

🏗

Too Light for Enterprise

Open-source chat UIs lack multi-tenancy, audit trails, guardrails, and the operational discipline required for regulated environments.

Features

Everything you need.
Nothing you don't.

☸

Kubernetes Native

Container-per-agent isolation. CRDs, operator, Istio mTLS. Agents are first-class K8s citizens.

🧪

Evaluation-Driven

14 MLflow GenAI scorers. LLM judges gate every deployment. No agent reaches production without passing.

🛡

Governance & Compliance

Scorecards across quality, safety, cost, compliance. Guardrail policies. Full audit trail.

💾

Git-Backed Versions

Immutable YAML artifacts with SHA-256 checksums. Every version reproducible. Instant rollback.

🤖

Multi-Model

Ollama, OpenAI, Anthropic, Vertex AI, Bedrock, Google AI. Switch models without changing code.

🔧

MCP & Skills

HTTP, CLI, MCP tools. Anthropic SKILL.md format. Extensible tool ecosystem.

🚦

Canary Deployments

Flagger + Istio progressive delivery. Quality gate webhooks. Auto-promote or auto-rollback.

📊

Real-Time Monitoring

AI Radar dashboard. MLflow tracing with token usage. Per-agent health, latency, cost tracking.

How It Works

Three steps to production

1

Define

Configure your agent: model, prompt, tools, skills, guardrails. Deploy to Kubernetes with one click.

2

Evaluate

LLM judges score safety, correctness, relevance. Quality gates block bad versions automatically.

3

Monitor

Production traces, user feedback, auto-scoring. Negative feedback becomes new test cases. The system improves itself.

# Deploy an agent with evaluation gate
curl -X PATCH http://recif:8080/api/v1/agents/my-agent/config \
  -H "Content-Type: application/json" \
  -d '{
    "modelType": "anthropic",
    "modelId": "claude-sonnet-4-20250514",
    "systemPrompt": "You are a helpful assistant.",
    "config": {
      "governance": {
        "min_quality_score": 75,
        "risk_profile": "high"
      }
    }
  }'

# Release auto-evaluates with 6 LLM judges
# Score >= 75 → deployed. Score < 75 → rejected + rolled back.
